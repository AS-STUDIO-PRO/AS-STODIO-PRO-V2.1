AS-STUDIO-PRO V2.1 is an optimized, modular, and production-ready implementation of **Wav2Lip-GAN** integrated with an interactive **Gradio Web UI**. It solves common checkpoint corruptions, dependency mismatches, and Librosa/PyTorch version errors out of the box, with native **1080p HD Resolution** rendering.

## 🛠️ Features & Setup
- **Dual-Cell Architecture:** Separate setup and execution cells to prevent checkpoint losses.
- **Full 1080p Rendering:** Video processing outputs locked to High-Definition 1080p.
- **Terminal Console Logging:** Real-time progress output.
- **Aspect Ratio Control:** 16:9, 9:16, 1:1, or Original.
